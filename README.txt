"20150814 All PNAS2011 data stitched (auto-mirrored, fixed, exported).mat" 
contains tracking data for all 12 worms in the PNAS2011 dataset, including 
corrections for dorsal/ventral orientation, etc.

Format is a cell array of 12 cells, each containing an Nx5 matrix, where a row 
is a time point, and each eigenmode has a column.

For reference, the "EigenWorms.mat" file contains the eigenworms used in this 
analysis.

GJS-Note (Aug 7 2016) All mode traces are downsampled to 16 fps.
